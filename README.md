# Salesforce Log Inspector

A Chrome extension that helps you inspect, analyze, and optimize your Salesforce Debug Log.

- 🗂️ Log Explorer
- 🐞 Apex Debug
- 🏳️‍🌈 Execution Timeline
- 🔗 Execution Tree  
- 🔎 Execution Analysis  
- 🛢️ SOQL Analysis
- 🧮 DML Analysis
